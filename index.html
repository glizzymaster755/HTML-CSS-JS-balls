Html

HTML (HyperText Markup Language) is the code that is used to structure a web page and its content

  • a piece of markup language used to indicate the beginning and end of an HTML element in an HTML document
  • An opening tag begins a section of page content, and a closing tag ends it
  • a piece of markup language used to adjust the behavior or display of an HTML element
  • a component of an HTML document that tells a web browser how to structure and interpret a part of the HTML document
